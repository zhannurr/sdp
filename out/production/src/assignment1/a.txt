Assignment 1

Разработайте консольное приложение на языке Java, которое симулирует процесс заказа и приготовления кофе в кофейне. Приложение должно использовать следующие порождающие паттерны:
1.Одиночка (Singleton): Создайте класс, который управляет заказами и гарантирует, что существует только один экземпляр кофейни в приложении.
2.Фабричный метод (Factory Method): Реализуйте фабрику, которая создает различные виды кофе.
3.Абстрактная фабрика (Abstract Factory): Создайте абстрактную фабрику, которая производит семейства взаимосвязанных объектов ингредиентов для кофе (например, разные виды молока, сиропов, добавок). Реализуйте конкретные фабрики для разных типов кофе (классы, которые реализуют интерфейс).
4.Прототип (Prototype): Реализуйте возможность клонирования заказов кофе, чтобы клиент мог повторить предыдущий заказ без повторного ввода всех деталей. Метод clone() должен копировать не только основной объект кофе, но и его ингредиенты.
5.Строитель (Builder): Создайте класс, который позволяет клиенту шаг за шагом создавать кастомный кофе с выбором ингредиентов и настроек.
Требования:
Приложение должно быть написано на Java и запускаться в среде разработки IntelliJ IDEA.
Необходимо продемонстрировать использование всех пяти паттернов в коде.
Код должен быть структурирован, следовать соглашениям об оформлении кода и содержать комментарии.
В основном классе Main должна быть показана работа приложения и использование всех паттернов.
Критерии оценивания:
1.  Корректность (30%): Точное и правильное применение каждого из пяти порождающих паттернов.
2.  Качество кода (20%): Чистый, понятный и поддерживаемый код, соответствующий стандартам оформления Java.
3.  Демонстрация (20%): Эффективное и корректное использование всех паттернов в основном классе приложения.
4.  Документация (15%): Наличие комментариев, объясняющих реализацию и ключевые моменты кода.
5.  Полнота (15%): Выполнение всех требований задания, включая размещение проекта на репозитории GitHub, и отсутствие ошибок при компиляции и запуске приложения.
